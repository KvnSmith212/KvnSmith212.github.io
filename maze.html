Maze Generation (Java)

This application gives the user several different algorithms to create mazes, with each algorithm offering a different style. It also includes a solve method, using a depth first search approach.

View the code here.

Mazes being generated in order: Binary, Depth First, Ellers, Hunt and Kill, and Randomized Prim

Mazes can also be made much larger (or smaller) and solved using a depth first approach.
